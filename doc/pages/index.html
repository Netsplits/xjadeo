XJADEO - The X Jack Video Monitor

http://xjadeo.sf.net/

Xjadeo is a software video player that displays a video-clip in sync with an external time source (MTC, LTC, JACK-transport). Xjadeo is useful in soundtrack composition, video monitoring or any task that requires to synchronizing movie frames with audio events.
